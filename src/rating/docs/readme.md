Rating directive that will take care of visualising a star rating bar.

## Usage

`m.ui.rating(settings)`

### Settings

* `iconOff`: Unrated icon.

* `iconOn`: Rated icon.

* `max`: Rating scale (default: 5).

* `rating`: Default rating (Default: 0).

* `hover`: Hover or not.

* `readOnly`: Read only or not.
